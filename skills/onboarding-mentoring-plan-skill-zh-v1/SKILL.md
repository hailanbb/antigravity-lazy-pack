# 新员工入职带教 Skill

## 何时使用

当用户需要处理“新员工入职安排、岗位学习计划、带教任务拆解、试用期跟进”相关任务时使用本 Skill。

## 输入材料

- 原始业务资料：聊天记录、表格、录音转写、照片说明或文档片段。
- 业务规则：判断标准、责任边界、风险等级和输出口径。
- 输出要求：需要给谁看、用于决策还是执行、是否需要复核。

## 工作流

1. 先识别资料类型，清理重复、缺失和无关信息。
2. 按模板抽取关键字段，形成结构化中间表。
3. 使用规则库判断优先级、风险点、责任人和下一步动作。
4. 输出可直接使用的结果，并列出待人工确认事项。
5. 如果信息不足，先提问补齐，不要编造。

## 输出要求

- 必须包含摘要、关键证据、风险等级、行动项、责任人、截止时间和待复核事项。
- 不能替用户做不可逆承诺，涉及金额、录用、处罚、合规结论时必须标注“需人工确认”。
- 输出应使用简体中文，避免空泛建议。
