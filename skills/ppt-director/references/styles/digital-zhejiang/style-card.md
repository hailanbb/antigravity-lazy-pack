# Style Card: digital-zhejiang

## Positioning

政务数字化、国企技术运营、智慧城市、区域治理和数字改革汇报风格。整体气质正式、可信、清爽，偏政务蓝白视觉。

## Canvas

- 16:9 宽屏。
- 参考渲染画布：1280 x 720 px。

## Visual DNA

- 正文页：白底、大留白、左上浅蓝几何标题底、右上蓝色品牌角标、右下浅蓝城市剪影。
- 封面和结束页：深蓝满版几何叠层背景，中央白色大标题。
- 组件：直角矩形、蓝橙标题条、浅蓝虚线框、蓝/橙圆形节点、蓝橙箭头。
- 信息表达：结构图、流程图、目录、分栏说明、图标圆形节点。

## Color System

| Role | Color | Usage |
| --- | --- | --- |
| 主色蓝 | `#014EC6` | 页面标题、编号、主节点、图标圆底、重点数字、标题条、实线边框 |
| 辅助橙 | `#FFAA11` | 次级模块、强调箭头、辅助节点、对比组、提示备注 |
| 亮蓝 | `#0065E2` | 细线、连接线、局部强调 |
| 正文黑 | `#000000` / `#262626` | 正文、说明文字、目录文字 |
| 浅蓝背景 | `#E7EFFD` | 普通信息背景、虚线框、城市剪影 |
| 浅灰背景 | `#F9F9F9` | 图片占位、普通内容区背景 |
| 白色 | `#FFFFFF` | 深蓝页文字、圆形节点内文字、卡片背景 |

正文页推荐比例：白色 70%-85%，主蓝 10%-20%，橙色 3%-8%。

## Typography

- 主字体：微软雅黑。
- 封面主标题：44-60 pt，Bold，白色。
- 正文页标题：24-36 pt，Bold，主蓝。
- 大标题：36 / 44 / 60 / 72 pt，Bold。
- 小标题：20 / 24 / 28 pt，Bold。
- 正文：16 / 18 pt，Regular 或 Bold。

Avoid:

- 微软雅黑 Light。
- 宋体。
- 特殊字体。

## Fixed Elements

- 左上标题区：x 约 60 px，y 约 20 px。
- 右上角标：贴右上，蓝色横条或斜切块。
- 右下城市剪影：固定资产 `references/styles/digital-zhejiang/assets/city-skyline.png`，x 约 760 px 起，y 约 515 px 起，锚定右下，保持宽高比和透明背景。
- 封面品牌标识：顶部中轴或偏上。

## Do

- 保持政务蓝白秩序感。
- 每页只表达一个主结构。
- 用蓝色表达主链路，橙色表达辅助链路。
- 标题要短、实、前置结论。
- 数据、流程、结构优先于装饰。

## Don't

- 不要大面积紫色、红色、绿色。
- 不要复杂阴影、圆角卡片堆叠、花哨渐变。
- 不要互联网营销式 hero 或过度视觉化。
- 不要把橙色变成主视觉。
- 不要使用手绘感或娱乐化图标。
