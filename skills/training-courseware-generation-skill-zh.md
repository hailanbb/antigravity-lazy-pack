# 培训课件生成 Skill

## 适用场景
很多企业不缺文档，缺的是能让新人快速学会的培训材料。文档写得越全，新人越不知道重点在哪里。培训课件生成 Skill 的价值，是把内部资料重新组织成一套有目标、有路径、有练习、有考核的课程包。

## 输入材料
- **原始资料**：聊天记录、文档、表格、截图说明、问卷、转写文本或业务记录。
- **背景信息**：任务目标、对象范围、时间范围、当前约束。
- **输出偏好**：表格、清单、报告、课程包或可复制到协作工具的格式。

## 工作流
1. 先识别任务目标，不直接输出结论。
2. 按模板抽取关键字段，缺失信息必须列为待确认项。
3. 对信息进行分类、分级和优先级判断。
4. 输出可执行清单，包含负责人、动作、截止时间和验证方式。
5. 最后给出复盘建议，方便下一次复用。

## 输出要求
- 使用简体中文。
- 先给结论摘要，再给结构化表格。
- 风险、优先级、待确认事项必须单独列出。
- 不确定的信息不能编造，必须标注“待确认”。
